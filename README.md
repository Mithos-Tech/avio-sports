# Avio Sports | Movilidad Eléctrica de Alto Rendimiento

Avio Sports es una plataforma de vanguardia dedicada a la movilidad eléctrica de lujo y alto rendimiento. Diseñada con una estética minimalista y sofisticada, la aplicación ofrece una experiencia inmersiva para explorar el futuro de la conducción sostenible.

## 🚀 Características

- **Diseño Inmersivo**: Interfaz de usuario sofisticada con efectos de profundidad 3D y animaciones fluidas.
- **Catálogo Interactivo**: Exploración detallada de modelos eléctricos de alto rendimiento.
- **Ecosistema de Servicios**: Sección dedicada a financiación, venta, alquiler premium y servicio técnico.
- **Optimización SEO**: Configuración profesional para motores de búsqueda y redes sociales.
- **Responsive Design**: Experiencia optimizada para dispositivos móviles, tablets y escritorio.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: React 19 + TypeScript
- **Estilos**: Tailwind CSS 4
- **Animaciones**: Motion (Framer Motion)
- **Iconos**: Lucide React
- **Build Tool**: Vite 6
- **Hosting Sugerido**: Vercel / Netlify / GitHub Pages

## 💎 Créditos y Recursos

Este proyecto utiliza los siguientes recursos de alta calidad:

- **Imágenes**: [Cloudinary](https://cloudinary.com/), [Unsplash](https://unsplash.com/), [Picsum Photos](https://picsum.photos/)
- **Tipografía**: [Google Fonts](https://fonts.google.com/) (Syncopate, Space Grotesk)
- **Iconografía**: [Lucide React](https://lucide.dev/)
- **Hosting de Activos**: Cloudinary (Logos y multimedia optimizada)

## 📦 Instalación y Desarrollo

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/avio-sports.git
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Construye para producción:
   ```bash
   npm run build
   ```
