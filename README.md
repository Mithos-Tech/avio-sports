# Avio Sports | Movilidad Eléctrica de Alto Rendimiento

> **Un proyecto de Inspyrio** > Concepto de plataforma premium enfocada en la exhibición y venta de vehículos eléctricos en tendencia mundial.

Avio Sports es una propuesta de vanguardia dedicada a la movilidad eléctrica de lujo. Diseñada bajo el sello de **Inspyrio**, esta aplicación ofrece una experiencia inmersiva para explorar el futuro de la conducción sostenible, combinando una estética minimalista con un rendimiento técnico superior.

## 🎯 Propósito del Proyecto
Este proyecto fue creado para demostrar la capacidad de **Inspyrio** en el desarrollo de interfaces de alto impacto para el sector automotriz, priorizando la velocidad de carga, la gestión eficiente de activos digitales y una experiencia de usuario (UX) de nivel premium.

## 🚀 Características

- **Diseño Inmersivo**: Interfaz sofisticada con efectos de profundidad y animaciones fluidas mediante Motion.
- **Catálogo Interactivo**: Exploración detallada de modelos eléctricos de alto rendimiento.
- **Ecosistema de Servicios**: Secciones estratégicas para financiación, venta, alquiler y servicio técnico.
- **Optimización SEO**: Configuración profesional para visibilidad en motores de búsqueda.
- **Responsive Design**: Adaptabilidad total para móviles, tablets y escritorio.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: React 19 + TypeScript
- **Estilos**: Tailwind CSS 4
- **Animaciones**: Motion (Framer Motion)
- **Iconos**: Lucide React
- **Build Tool**: Vite 6
- **Asset Management**: Cloudinary (Optimización de imágenes y logos)
